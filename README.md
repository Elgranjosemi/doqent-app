# Doqent - App de Registro de Entregas

Aplicación web progresiva (PWA) para registrar entregas con firma digital, búsqueda, filtros y generación de PDFs.

## Funcionalidades

- Registrar entregas con firma
- Gestionar compañeros
- Historial con búsqueda y filtros
- Generación de PDFs
- Instalable como app móvil

## Cómo usar

1. Abre `index.html` en tu navegador
2. Registra compañeros
3. Registra entregas
4. Genera PDFs
5. Instala como app en tu móvil (menú de Chrome/Safari → Agregar a pantalla de inicio)

## Tecnologías usadas

- HTML, CSS, JavaScript
- IndexedDB (simulado con localStorage)
- signature_pad.js
- jspdf.umd.min.js

- html2canvas.min.js
